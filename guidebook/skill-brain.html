Docs/Improve/tcx-brain

Scoped task

tcx-brain

Author and manage Investment Brain sources, immutable installed versions, activation state, rollback, and removal through one explicit entrypoint.

Manage a user-owned source #

Create, inspect, revise, validate, or explicitly delete a workspace-local source under investment-brains/<investment-brain-id>. Start a new trading-research task whose exact first line is $tcx-brain; do not wrap it in $tcx-build. Authoring uses only user-selected lessons and counterexamples, performs privacy review, and abstracts general inquiry and interpretation heuristics instead of copying private cases.

Create a source

$tcx-brain
Create a local Investment Brain from these selected lessons and counterexamples.

When the request contains enough source material and the action is reversible, the skill can create or revise the draft without an extra confirmation. Deletion and missing privacy choices remain explicit.

Use the managed lifecycle #

The same scoped task can list, inspect, validate, install, update, activate, deactivate, roll back, or remove a Brain through the proof-protected manage_investment_brain service. Source authoring stays a normal workspace-file operation, while registry and projection changes go through that MCP tool. The Research task never reopens the generated launcher, attached runtime, database, or credentials.

Install a reviewed source

$tcx-brain
Validate and install investment-brains/investment-brain-quality-growth as inactive.

Installation starts inactive. Activation requires an explicit request. Managed remove drops the Head Manager projection but retains immutable versions for historical provenance; deleting the user-owned source is a separate action.

Select a Brain only for the run that needs it #

Use an active Brain

$investment-brain-quality-growth
Analyze ACME and state the evidence that would falsify the framework.
tcx-brain manages Brains; it does not select one for analysis. A Brain never selects roles, calls tools, changes policy, edits memory, or grants approval, broker, order, or execution authority.
