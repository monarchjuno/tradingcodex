Skill directory

All user-facing skills

TradingCodex has 13 user-facing skills: ten primary entrypoints and three supporting skills. Role-owned skills are intentionally absent from this directory because tcx-workflow selects them only when specialist work is needed.

Research #

01tcx-workflowStart a dynamic investment research workflow.→02tcx-planClarify an outcome, scope, action boundary, and stop condition.→03tcx-automateCreate a scheduled task that invokes the actual work skill.→

Order #

New to broker setup? Start with From provider to order before choosing a final-action entrypoint.

04tcx-order-allowAdmit one possible later order attempt in the current root turn.→05tcx-order-submitSubmit one already-approved order with exact identifiers.→06tcx-order-cancelCancel one known submitted broker order with exact identifiers.→

Improve #

07tcx-memoryRetrieve, replay, review, and validate past judgment.→08tcx-brainAuthor and manage Investment Brain plugins.→09tcx-strategyAuthor reusable user decision rules.→

Customize #

10tcx-buildOpen an explicit managed workspace Build turn.→11tcx-investor-contextPreview workspace-local suitability context.→

Help & status #

12tcx-dashboardOpen the read-only dashboard in Codex by default.→13tcx-serverCheck viewer, service, update, and MCP readiness.→
